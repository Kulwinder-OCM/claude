Flask App Deployment Guide

Deploy your AI Brand Analysis app to a Python-compatible platform

Netlify Limitation

Netlify is designed for static sites and JavaScript functions. Python Flask applications need a different hosting platform.

Vercel (Recommended)

Supports Python functions with minimal configuration.

  1. Connect your GitHub repo to Vercel
  2. Vercel auto-detects Python
  3. Add environment variables
  4. Deploy automatically
Deploy to Vercel
Railway

Full Flask application hosting with database support.

  1. Connect GitHub repository
  2. Railway detects Flask app
  3. Configure environment variables
  4. One-click deployment
Try Railway
Render

Python web services with free tier available.

  1. Create Render account
  2. Connect GitHub repo
  3. Set build/start commands
  4. Configure environment
Use Render
Local Development

Run the Flask app on your local machine.

cd Claudelife-main
pip install -r requirements.txt
python app.py

Visit: http://localhost:5001

Environment Variables Required

Make sure to set these environment variables on your hosting platform:

  • CLAUDE_API_KEY - Your Anthropic Claude API key
  • GEMINI_API_KEY - Google Gemini API key
  • SCREENSHOT_API_KEY - ScreenshotOne API key
  • AI_TEXT_ANALYSIS_PROVIDER=claude
  • AI_TEXT_GENERATION_PROVIDER=claude
  • AI_WEB_ANALYSIS_PROVIDER=claude
  • AI_CONTENT_STRATEGY_PROVIDER=claude
